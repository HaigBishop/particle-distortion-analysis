[size=20dp]Application Description[/size]
Particle Distortion Analysis (PDA) is software for the analysis of microaspiration (or ion pipette aspiration) data.
The source code, along with instructions, documentation and a Windows executable, is available on Github.

The Github repository: [ref=https://github.com/HaigBishop/particle-distortion-analysis][color=3498db]github.com/HaigBishop/particle-distortion-analysis[/color][/ref]
